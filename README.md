Projeto Integrador referente ao Projeto Notificar, um aplicativo de notificações de ocorrências em estacionamentos ou no trânsito.

Com ele você pode contactar um motorista para reportar incidentes como janela aberta, estacionamento inadaqueado, alarme disparou e etc

A identificação da conta de cada motorista é pela placa de motorista, como se fosse o @username.

O aplicativo tem outras funções dedicadas para condomínios e estacionamentos privados que podem enfatizar regras ou facilmente identificar motoristas em seus respectivos
estacionamentos e entrar em contato caso necessário.
